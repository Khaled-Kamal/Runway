# 🛍️ Ranway – Fashion Store App (Flutter)

**Ranway** is a modern fashion store mobile application built with **Flutter**.
It delivers a smooth and stylish shopping experience with clean UI, seamless navigation, and interactive animations.

---

## ✨ Features

* 🎬 **Animated splash screen**
* 📽️ **Interactive video background** on the homepage
* 👗 **Multiple product categories**: Men, Women, Kids, Deals, Health
* 🛒 **Product listing** with `GridView`
* 🧾 **Product details screen** featuring:

  * High-quality product image
  * Price and likes
  * Color & size selection
  * **"Add to Bag"** button
* 💡 **Bottom sheet interaction** for product info
* 🧩 **Modular architecture** with reusable custom widgets
* 🖋️ Integrated with **Google Fonts** for modern typography
* 🎨 **Clean, responsive, and user-friendly design**
